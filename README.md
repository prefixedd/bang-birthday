# 🎂 Happy Birthday Website

A personalized animated birthday website built with pure HTML, CSS, and JavaScript. Hosted on GitHub Pages.

🌐 **Live Site:** [prefixedd.github.io/1](https://prefixedd.github.io/1)

---

## ✨ Features

- Animated birthday greeting with letter-spaced text effects
- Interactive cake page (`cake.html`) that opens on click
- Floating balloons and decorative elements
- Embedded background music (YouTube, autoplayed on loop)
- Custom images — photos, GIFs, hearts, flowers, hats, and more
- Link to wish Bang directly on Instagram
- Fully responsive design

---

## 📁 Project Structure

```
/
├── index.html       # Main birthday landing page
├── cake.html        # Animated cake reveal page
├── style.css        # All styling and animations
└── Images/          # All local assets (photos, GIFs, decorations)
```

---

## 🚀 Running Locally

No dependencies or build steps needed — just open in a browser:

```bash
git clone https://github.com/prefixedd/1.git
cd 1
# Open index.html in your browser
```

Or simply visit the live site at [prefixedd.github.io/1](https://prefixedd.github.io/1).
